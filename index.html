✦ How It Works

Three Ways to Log Your Server

Choose the setup that fits your server — from a single catch-all channel to granular per-event logging.

📋
Default Channel

Route all 85+ events into a single unified log channel. Perfect for smaller servers that want everything in one place.

🗂️
Category Channels

Separate channels per category — Member logs go here, Moderation logs go there. Clean, organized, readable.

🎯
Event-wise Channels

Maximum control. Assign each individual event to its own channel. Ideal for large servers with strict audit needs.

🚫
Disable Any Event

Don't want reaction logs cluttering things up? Turn off any individual event with a single command. Full control, no clutter.

⚡
Real-time Logging

Events are captured and sent to your channels instantly. Nothing slips through the cracks, even at high server activity.

🔒
Admin Controls

All configuration is restricted to admins. Set it once, let it run — your settings are saved securely in the database.

🔀
Mix & Match Modes

Set a default channel, then override specific categories or events. The system resolves the right channel automatically.
